## 升级指南

### 从 v0.1.4 升级到 v0.1.5
1. 备份当前配置
2. 卸载旧版本: `pip uninstall nonebot-plugin-arkguesser`
3. 安装新版本: `pip install nonebot-plugin-arkguesser==0.1.5`
4. 重启机器人
